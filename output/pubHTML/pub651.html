Alan Winfield left this review about The Sportsman

The Sportsman is a very large pub that is set in back streets off Gloucester road.
Once inside the main drinking room is L shaped with the left side extending to the rear,the room is carpeted and has normal tables and chairs,there is a small room behind the bar in the main room which had a covered up pool table,off this room is a very large room which has six pool tables most were in use.
There are also rooms upstairs which were in use but i did not go up to look at these.
There were TVs around the main room showing a Aston Villa v Chelsea game.
Only one real ale on the bar which was Doom Bar,this was as good as it gets.
The pub was very busy with mainly Bristol Rovers fans,but with it being so large i got a seat without any problems.
This is a decent enough pub especially if you fancy a game of pool.

Pub visited 2/4/2016
