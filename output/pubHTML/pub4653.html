Michael Wright left this review about Kings Arms Hotel

Corner position pub / hotel in centre of Lostwithiel, a handy start or finish point for the other 4 pubs in town

Large L shaped bar, one arm of which is pool table space. When I visited it was having a mini make over, so should be clean & tidy [it was not bad anyway]

Casque marked ales, St Austell brewery linked, I stuck to the Tribute which suits my taste, but did venture onto Cornish Rattler at evenings end

I am told it serves food, but lunchtime only [chip shop up the road], so unable to comment on this aspect

Has B & B [see web site]

I found the atmosphere convivial & friendly, it seems to be a local to many as most seemed to know one another, which maybe not be hard in Lostwithiel

I did go back the following evening, it was just as nice

5 pubs in Lostwithiel, if Globe is shut or full, this is next, but no point in walking past here to the Globe, try this first before walking on

The web site says walking distance from station, whilst factually true I would not advise this with a suitcase or two, it is not "just a few minutes up the road"

On 1st June 2010 - rating: 7
